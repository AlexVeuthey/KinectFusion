Agglomerative Information Bottleneck (AIB)
Author:
Brian Fulkerson
Andrea Vedaldi

aib.h implemens the Agglomerative Information Bottleneck (AIB) algorithm as first described in [11] .

AIB takes a discrete valued feature and a label and gradually compresses by iteratively merging values which minimize the loss in mutual information .

While the algorithm is equivalent to the one described in [11] , it has some speedups that enable handling much larger datasets. Let N be the number of feature values and C the number of labels.
